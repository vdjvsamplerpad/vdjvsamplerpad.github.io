Install VDJV Sampler Pad on Android

You are already on the install page. Follow these browser steps.

  1. Tap the browser menu in Chrome or Edge.
  2. Tap "Install app" (or Add to Home screen).
  3. Confirm Install when prompted.
  4. Open the installed icon from your launcher. It opens at vdjvsamplerpad.online/.

Preparing install prompt...

If install is not shown in the menu, refresh this page and wait a few seconds before opening the menu again.
